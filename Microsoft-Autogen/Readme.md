#### AI Agents and Agentic AI Systems Via Microsoft Autogen  
- Install Anaconda which is is a data science distribution that bundles the Python language, a package manager called Conda, and a collection of pre-installed libraries.
- Create a folder for projects and create a environment with:
   - conda create -p venv python=3.12  //in the location of terminal
   - pip install -r requirements.txt  //we created a file requirements.txt we can define required libraries to get installed during project creation like pandas, numpy.
